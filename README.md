# Overview
[こちら](http://shira2paper.blog.fc2.com/blog-entry-6.html)を参考に修正した、
Firefox の add-on "Qucik Drag" です。

# Install
about:configにて
```
xpinstall.signatures.required
```
をfalseに変更してください。
リポジトリトップにあるquickdrag-improved.xpiをFirefoxのD&Dしてください。

# Usage
QuickDragと同様です。
